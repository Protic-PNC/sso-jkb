## Run Locally

Clone the project

```bash
  git clone https://github.com/Protic-PNC/sso-jkb
```

Go to the project directory

```bash
  cd sso-jkb
```

Install dependencies

```bash
  npm install
```

Migrate DB 

```bash
  php artisan migrate:fresh (or) php artisan migrate
```
Db seed 

```bash
  php artisan db:seed
```
Run 

```bash
  php artisan serve
```
